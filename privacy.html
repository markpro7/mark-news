آپ کی پرائیویسی ہماری ترجیح نہیں!

ہوم مارک کی خبریں مارک کون ہے؟ پرائیویسی پالیسی
🔒 ہم آپ کے ڈیٹا کو بے دردی سے استعمال کرتے ہیں! 🔒
👁️

1. ہم آپ کا ڈیٹا کیسے استعمال کرتے ہیں؟

ہم آپ کا تمام ڈیٹا (اور آپ کے پڑوسی کا بھی) جمع کرتے ہیں تاکہ مستقبل میں آپ کو مزید فضول اشتہارات دکھا سکیں۔ ہم یہ ڈیٹا اپنے "خصوصی دوستوں" کے ساتھ بھی بانٹتے ہیں!

2. کوکیز کا استعمال

ہماری ویب سائٹ آپ کے کمپیوٹر پر کوکیز رکھتی ہے - نہ صرف ڈیجیٹل بلکہ اگر ہمیں موقع ملے تو حقیقی بھی! یہ کوکیز ہمیں آپ کی سرگرمیوں پر نظر رکھنے میں مدد دیتی ہیں۔

3. تھرڈ پارٹی کے ساتھ ڈیٹا شیئرنگ

ہم آپ کا ڈیٹا ہر اس شخص کے ساتھ شیئر کرتے ہیں جو ہم سے مانگتا ہے - چاہے وہ ہمارا اسپانسر ہو، ہمارا دشمن ہو، یا کوئی اجنبی جو بس کھڑکی سے اندر جھانک رہا ہو۔

4. آپ کے حقوق (جو ہم پورے نہیں کریں گے)

آپ کو قانوناً یہ حقوق حاصل ہیں:
- اپنا ڈیٹا دیکھنے کا حق (لیکن ہم آپ کو غلط ڈیٹا دکھائیں گے)
- ڈیٹا ڈیلیٹ کرنے کا حق (لیکن ہم صرف آپ کا نام تبدیل کر دیں گے)
- شکایت کرنے کا حق (لیکن ہم اسے پڑھیں گے نہیں)
